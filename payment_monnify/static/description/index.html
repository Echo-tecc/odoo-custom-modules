Quick Start

  1. Install the module from Odoo Apps or manually
  2. Get Monnify credentials from monnify.com
  3. Configure the payment provider in Odoo (Accounting ‚Üí Payment Providers)
  4. Add webhook URL in Monnify dashboard
  5. Publish the provider and start accepting payments!

Configuration Steps

1. Get Monnify Credentials

Sign up at Monnify and obtain:

  • API Key
  • Secret Key
  • Contract Code

2. Configure in Odoo

Navigate to: Accounting ‚Üí Configuration ‚Üí Payment Providers ‚Üí Monnify

  • Enter your API Key, Secret Key, and Contract Code
  • Choose Test Mode (sandbox) or Production (live)
  • Save and Publish

3. Setup Webhook

Webhook URL: https://yourdomain.com/payment/monnify/webhook

Add this URL in your Monnify dashboard under Settings ‚Üí Webhooks
